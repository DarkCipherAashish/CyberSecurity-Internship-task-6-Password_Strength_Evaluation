## âš ï¸ Password Attack Methods & Defense

| Attack Type | Description | Prevention |
|---|---|---|
| Brute Force | Tries all possible combinations | Longer and complex passwords |
| Dictionary Attack | Uses common words and leaked lists | Avoid real words and predictable patterns |
| Phishing | Tricks users into revealing login details | Verify links & emails before entering credentials |
| Credential Stuffing | Tries stolen passwords on other platforms | Use unique passwords + MFA |

## âœ… Conclusion

The testing clearly shows that **password length, randomness, and character variety** have a major impact on security strength.  
Simple passwords are easy to crack, while **long, mixed-character passphrases** achieve the highest strength scores.

Combining strong passwords with **MFA** provides an extra layer of protection.
